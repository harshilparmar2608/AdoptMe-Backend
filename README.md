# AdoptMe-Backend

## Project Overview
AdoptMe-Backend is a backend service for the Adopt Me application, aimed at facilitating the adoption of pets through a user-friendly interface. The service provides necessary data management, including user accounts, pet listings, and adoption requests.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/<your_username>/AdoptMe-Backend.git
   cd AdoptMe-Backend
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Set up environment variables:**
   Copy the `.env.example` to `.env` and fill in the required details.
   ```bash
   cp .env.example .env
   ```
4. **Run the application:**
   ```bash
   npm start
   ```
5. **Access the API:**
   The backend service runs on `http://localhost:3000` by default.
